Overall Program

This page was developed to help familiarize new Staples Business Advantage members' with the membership program. It uses the Vidyard Video Player API to deliver video. This page is particularly interesting because I developed it to so that when the user hits "Watch Video" trigger, the video slides into view and begins playing and the page scrolls into focus for the video. If the window width is less than 640 pixels, and should the user scroll and the video is no longer in the viewport, the video pauses. Should the user scroll the video back into the viewport the video begins to play again.

STACK: HTML5, SASS, CSS3, Transitions, Javascript, JQuery, Bootstrap, Vidyard API
